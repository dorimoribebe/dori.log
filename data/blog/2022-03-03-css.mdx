# 🎈 CSS란
> Cascading Style Sheets 
- 계단식으로 스타일을 정의하는 문서
- HTML문서에 스타일을 더해주는 역할 
---   
# 🎈 CSS 문법
## 💻 기본 문법
```
선택자{
   속성명: 속성값;
}
```
- 예시
```css
p{
   color: red;
   /*p태그 안의 폰트 색을 빨간색으로 지정하겠다는 뜻*/
}
```
---   
## 💻 코드 적용 방식
### 📎 인라인 스타일
```html
<p style="color: blue;">
   인라인 스타일로 적용
</p>
```
- 태그에 style 속성 추가, 요소에 직접적으로 스타일 정의
- 선택자 필요 없음
- 웹 콘텐츠와 스타일 시트 분리 어려움

### 📎 스타일 태그
```html
<style>
   p{
      color: red;
   }
</style>
```
- html 문서의 `<head></head>` 태그 안에 작성

### 📎 문서 간의 연결
```html
<link href="css파일 경로" rel=" stylesheet">
```
- html 문서의 `<head></head>` 태그 안에 작성
- `rel` : 현재 문서와 외부 소스 연관 관계 기술하는 속성

---   
## 💻 선택자
- 선택자가 겹치는 경우 나중에 작성된 스타일이 적용
- 선택자는 다르나 요소가 겹치는 경우 우선순위는 아이디 > 클래스 > 태그

### 📎 기본 선택자
#### 📼 모든 요소 선택자
- `*` -> 모든 요소에 속성값 부여
```css
*{
   color: blue;
}
```
   
#### 📼 태그 선택자 
- 해당 태그에 속성값 부여
```css
p{
   color: blue;
}
```   

#### 📼 클래스 선택자 `.`
- 해당 클래스 속성값을 가진 요소에 속성값 부여
```css
.class{
   color: blue;
}
```

#### 📼 id선택자 `#`
- 고유한 식별자 역할을 하는 전역 속성
```css
#id{
   color: blue;
}
```   

### 📎 그룹 선택자
- 다양한 유형의 요소를 한꺼번에 선택하고자 할 때 사용
```css
p, h1, div{
   color: blue;
}
```
