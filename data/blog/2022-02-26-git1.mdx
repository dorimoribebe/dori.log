# 🎈 Git 시작하기
## 1. git 사용자 설정

```bash
git config --global user.name "dori"

git config --global user.email "dorimori_bebe@naver.com"
```

## git carriage-return auto 설정

`core.autocrlf true`

## 2. git 초기화

해당 디렉토리에서  `git init` → .git 파일 생성됨

## 3. git 파일 삭제

`rm -rf .git`

### git staged file 확인

 `git status` : staged file 확인

 `git add 파일명` : staged file에 파일 추가

git untracked 파일 전체 삭제(실제 파일 삭제됨) → `git clean -f`

## 4. 커밋하기

### 명령어
- `git commit`
- `git commit -m "commit message"`
- `git commit -am "commit message"`

### 커밋메시지
- 현재형, 동사형으로
- 해당하는 내용만 ex) login page 수정하면 login page수정한 내용만 커밋

### 로그 확인
- `git log`

### 커밋 취소 (이전 상태로)
- 이력 제거
`got reset {커밋아이디} --hard`

- 이력 유지
`git revert {커밋아이디}`

# 🎈 cmd 기본 명령어

```bash
# 현재 디렉토리 확인
pwd

# 현재 위치 기준 'project'라는 디렉토리로 이동
cd project

# 현재 위치 기준 상위 디렉토리로 이동
cd ../

# 현재 디렉토리
cd ./

# 파일 내용 확인
cat 파일명.확장자

# 파일 내용 확인 및 수정
vim 파일명.확장자

# 디렉토리 생성
mkdir 디렉토리이름

# 현재 디렉토리 파일 확인
ls -al

# 파일에 내용 추가
echo 내용 >> 파일명

# 빈 파일 생성
touch 파일명

# 파일 삭제
rm 파일명
```

# 🎈 git 연결 오류로 인한 해결
1. 로컬 저장소의 git 히스토리 삭제
(기존 github와의 연결 끊기)   
`rm -rf .git`

2. 로컬 저장소 초기화   
`git init`

3. 연결 확인   
`git remote -v`

4. 로컬 저장소의 파일 커밋   
`git add .`

5. 커밋   
`git commit -m "first commit"`

6. 푸시   
`git push origin master`
