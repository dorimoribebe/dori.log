# 🎈 .bashrc 수정하기

   git 환경변수를 설정해보겠다고 아무렇게나 `.bashrc`건드리다가 `vim`, `ls`와같은 기본 명령어들이 동작하지 않게 되었다,, 

## 1. 임시 환경변수 설정

```bash
export PATH=%PATH:/bin:/usr/local/bin:/usr/bin
```
이렇게 임시로 환경변수를 설정해서 기본 명령어를 사용할 수 있다!

## 2. `.bashrc` 파일 수정하기

```bash
vim .bashrc

# 이후 나오는 창에서 수정
```

## 3. `.bashrc` 등록하기

```bash
source .bashrc
```
