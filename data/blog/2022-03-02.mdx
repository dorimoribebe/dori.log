# 🎈 html 문법
## 🔅 태그
태그는 시작태그와 종료태그로 구성되어있음.
   
- 시작태그 `<태그이름>`
- 종료태그 `<\태그이름>`
   
시작태그와 종료태그 사이에 내용이 들어감
   
```html
<h1>this is title<\h1>
<p>this is paragraph<\p>
```

### 📎 기본 태그
- `<html></html>` : 웹페이지의 시작과 끝
- `<head></head>` : 웹페이지의 정보, 문서, 외부 파일 링크
- `<body></body>` : 브라우저에 표시되는 내용
- `<title></title>` : 문서 제목
- `<meta></meat>` : 인코딩, 키워드

### 📎 제목 태그
- `<h1></h1>`   
- `<h2></h2>`   
- `<h3></h3>`   
- `<h4></h4>`   
- `<h5></h5>`   
- `<h6></h6>`   

### 📎 줄바꿈 태그
- `<br>`

### 📎 구역 태그
- `<span></span>` : 줄바꿈 없음   
- `<div></div>` : 줄바꿈 1번
- `<p></p>` : 줄바꿈 2번   

### 📎 링크 태그
- `a` : 웹페이지, 외부 사이트 연결   
   ```html
   <a href="경로" title="링크 설명">내용</a>
   ```
- `link` : 외부 파일 연결
   ```html
   <link href='외부 css파일 경로' rel="stylesheet" type="text/css">
   ```
- `<img>` : 이미지 삽입
   ```html
   <img src="이미지 경로" alt="대체 텍스트">
   ```

### 📎 나열 태그
- `<ul></ul>` : unordered list. • 블릿 붙음
- `<ol></ol>` : ordered list. 숫자, 영문 순서 붙음
   사용 가능한 속성값들
      - `type="1"` : 숫자(default)
      - `type="a"` : 영어 소문자
      - `type="A"` : 영어 대문자
      - `type="i"` : 로마 소문자
      - `type="I"` : 로마 대문자
      - `start` : 순서대로
      - `reversed` : 역순서대로
- `<li></li>` : `<ul>`, `<ol>` 안에서 항목 나열

```html
<ul>
   <li>내용</li>
   <li>내용</li>
   <li>내용</li>
/ul>

<ol>
   <li>내용</li>
   <li>내용</li>
   <li>내용</li>
</ol>

<ol type="i" reversed>
   <li>내용</li>
   <li>내용</li>
   <li>내용</li>
</ol>
```

### 📎 강조 태그
- `<b></b>` : 굵게   
- `<i></i>` : 기울임체  
- `<u></u>` : 밑줄   
- `<s></s>` : 중간선
- `<strong></strong>` : 내용 강조 위해 진하게 표시
- `<em></em>` : 내용 강조 위해 기울임체

## 🔅 속성
- `id` : 고유한 속성 부여
- `class` : class 단위로 묶어서 속성 부여

```html
<h1 id="title">this is title<\h1>
<p id="context" class="main">this is paragraph<\p>
<p class="main">this is paragraph<\p>
<p>this is paragraph<\p>
```
