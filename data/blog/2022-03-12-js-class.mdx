# ğŸˆ getter & setter
```js
class Emp{
    constructor(name, age, job){
        this.name = name;
        this.age = age;
        this.job = job;
    }

    get age(){
        return this._age;
    }

    set age(value){
        this._age = value < 0 ? 0 : value;
    }
}

const emp1 = new Emp('dori', -100, 'frontend developer');
console.log(`${emp1.name}(${emp1.age}) is ${emp1.job}`);
```
> dori(0) is frontend developer
