# 🎈 font-family 
- 글꼴을 정의
- 글꼴명을 속성값으로 지정
- 여러개 글꼴 연달아 기입, 우선 순위 지정 가능
```css 
    *{
        font-family: monospace, serif;
    }
```

---   
# 🎈 font-size  
- px : 절대적 크기 픽셀
- rem : `<html>`태그의 최상위 요소에 대응하는 상대적 크기
- em : 상위 태그에 대응하는 상대적 크기
```html
    <head>
        <style>
            html{
                font-size: 16px;
            }

            div{
                font-size: 2rem;
            }

            span{
                font-size: 1.2em;
            }
        </style>
    </head>
    <body>
    폰트사이즈 16
    <div>
        폰트사이즈 16*2
        <span>
            폰트사이즈 16*2*1.2
        </span>
    </div>
    </body>
```

# 🎈 text-align
- 블록 내에서 텍스트의 정렬 방식 정의
```css
    .left{
        text-align: left;
    }

    .right{
        text-align: right;
    }

    .center{
        text-align: center;
    }

    .justify{
        text-align: justify;
    }
```
